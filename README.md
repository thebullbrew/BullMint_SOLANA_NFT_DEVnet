# BullMint-Solana-NFT-War-Room
Solana’s wallet adapter is the standard React wallet connection layer, and Metaplex’s Umi SDK is the modern JavaScript path for creating NFTs with Token Metadata.  


# Command to start:

npx create-next-app@latest bullmint

cd bullmint

npm install @solana/web3.js @solana/wallet-adapter-react @solana/wallet-adapter-react-ui @solana/wallet-adapter-base

npm install @metaplex-foundation/umi @metaplex-foundation/umi-bundle-defaults @metaplex-foundation/mpl-token-metadata


# First version features:

1. Connect Phantom/Solflare wallet
2. Show NFT card preview in browser
3. Input name, symbol, description
4. Upload image/metadata
5. Mint NFT on Solana Devnet
6. Show mint address + Solana Explorer link
